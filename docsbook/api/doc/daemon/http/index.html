Minimal HTTP server exposing daemon endpoints.

This module hosts a tiny Hyper-based server used exclusively for integration testing. It serves two endpoints:

  • /metrics – placeholder Prometheus metrics output.
  • /__health – liveness probe returning 200 OK with body ok.

The server runs on its own thread so that it doesn’t block the scheduler loop. It can be gracefully shut down using an oneshot channel.
